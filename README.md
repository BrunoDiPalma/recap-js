# 📝 To-Do List

Um projeto simples de lista de tarefas (To-Do List) desenvolvido com HTML, CSS e JavaScript puro.  
O objetivo foi praticar manipulação do DOM, eventos e uso do LocalStorage para persistência de dados.

---

## 🚀 Funcionalidades

- Adicionar tarefas
- Marcar tarefas como concluídas
- Remover tarefas
- Persistência de dados com LocalStorage (as tarefas não somem ao atualizar a página)

---

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- LocalStorage
